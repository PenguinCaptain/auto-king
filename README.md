# Auto King
A fully animated vue+gsap+threejs project for school assignment
https://penguincaptain.github.io/autoking
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

## How to use
1. Clone the repo
``` bash
git clone https://your-repo-url.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
